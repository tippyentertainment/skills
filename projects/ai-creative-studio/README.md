# AI Creative Studio

A web-based creative studio for generating AI content: images, videos, audio, voice, and 3D models.

## Features

- Dashboard interface with modern dark theme
- Image generation panel (text-to-image)
- Video generation panel (text-to-video)
- Audio/sound generation panel
- Voice generation panel (text-to-speech)
- 3D model generation panel
- Gallery view for generated content
- Export/download functionality
- History tracking

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
```

## API Integration

Connect to tasking.tech AI generation endpoints by updating the `handleGenerate` function in App.jsx.
